Abaixo os comandos para executar os três aplicativos:

Ordenação de cores
java -cp "lib/plot3d.jar;SOM.jar" exemplos.cores.OrdenaCoresExemplo 10

Aplicativo SOM 2D
java -cp "lib/plot3d.jar;SOM.jar" exemplos.grade2d.Grade2DExemplo 30

Aplicativo SOM 3D - Obs: "Pode arrastar e soltar para rotacinonar".
java -cp "lib/plot3d.jar;SOM.jar" exemplos.SOM3DEx1 30

Plot3D Exemplo - Obs: "Pode arrastar e soltar para rotacionar".
java -cp "lib/plot3d.jar;SOM.jar" exemplos.principal.PlanoCartesiano3DPrincipal

Atenção: para alterar o delay de cada iteração do algoritmo, basta alterar o valor após o comando que está definido como 30.
Caso não queira o delay, basta remover o parâmetro ou substituir o valor dele por zero.

Atenção: para executar os aplicativos pelos comandos acima, é necessário, pelo MS-DOS, navegar até a pasta onde está o 
arquivo "SOM.jar".
